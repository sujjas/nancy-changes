Custom Code

Add this to Framer Project Settings → Custom Code → Head to load the fonts.

<!-- Add to Project Settings → Custom Code → Head --> <link rel="preconnect" href="https://fonts.googleapis.com"> <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin> <link href="https://fonts.googleapis.com/css2?family=Newsreader:ital,wght@0,400;1,300;1,400&family=Atkinson+Hyperlegible:wght@400;700&display=swap" rel="stylesheet">
Two-font stack: With DM Sans replaced, you only need Newsreader (headings/display) and Atkinson Hyperlegible (everything else — body, buttons, nav, UI, eyebrows).
